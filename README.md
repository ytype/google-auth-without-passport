모듈 [googleapis](https://www.npmjs.com/package/googleapis)를 사용하여
passport를 사용하지 않고 google 인증 연습

## 설치

``` bash
npm install
```

## 시작

[google console](https://console.cloud.google.com/)에서 json파일을 발급받아서 google_key.json로 변경하면 됩니다.

``` bash
npm run start
```

3000번 포트로 서버가 시작됩니다.
